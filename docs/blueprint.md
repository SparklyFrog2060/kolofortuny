# **App Name**: SpinFlow

## Core Features:

- Wheel Creation & Management: Allows users to create, name, and delete multiple 'wheels of fortune', which act as containers for challenges.
- Challenge Item Management: Enables users to add, edit, and remove individual challenges or options for any selected wheel.
- Interactive Spin Interface: Presents a visual and interactive wheel that users can spin to randomly select a challenge.
- Spin Result Display: Clearly displays the challenge or option selected by the wheel after a spin is completed.
- AI Challenge Idea Generator: A generative AI tool that suggests new challenge ideas based on themes or keywords provided by the user, enhancing content creation.
- Data Storage (Firebase): Stores user-created wheels and their associated challenges persistently using Firebase Firestore.

## Style Guidelines:

- Primary color: Electric Violet (#A91AE5) to evoke creativity and dynamic engagement.
- Background color: Very light lavender (#F9F2FE) providing a clean and energetic canvas.
- Accent color: Vibrant Fuchsia (#F81FC6) used for calls to action and to draw attention to interactive elements.
- Headline and body font: 'Poppins' (sans-serif) for a modern, clean, and inviting feel, suitable for interactive elements and short descriptions.
- Utilize sleek, minimalist icons for wheel controls (spin, edit, delete) and challenge management, maintaining a cohesive aesthetic.
- A central, prominent area for the interactive wheel, flanked by collapsible panels or modals for wheel and challenge management.
- Smooth, engaging spinning animation for the wheel, and subtle celebratory effects upon revealing the selected challenge.
